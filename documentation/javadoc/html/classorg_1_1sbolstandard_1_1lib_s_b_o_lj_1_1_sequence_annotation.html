Detailed Description

The SBOL data model's SequenceAnnotation for RDF and Json.

Sequence Annotations give the location and direction of SequenceFeatures of a DnaComponent. The location is specified by the start and stop positions of the Sequence Feature along the DNA sequence. We are using the convention that the first base pair of a DNA sequence is position 1. This convention is established by the broader Molecular biology community, in literature. The direction of the feature is specified by the strand [+/-]. Sequences used are by convention assumed 5' to 3', therefore the + strand is 5' to 3' and the - strand is 3' to 5'.

void setId ( DnaComponent  parentDnaComp)

Creates an ID based on a hash of the start,stop,strand,and parent DnaCompoment.

Uses sha-256 to hash the information elements. The parentDnaComponent is used to make sure the hash is unique for any given Sequenceannotation of a DnaComponent. This also enforces the use of one SequenceAnnotation for a single position with strand.

\subsection{Detailed Description}
SBOL utils provide read and write methods for interacting with interfaces outside of libSBOLj. The utils include methods for writing SBOL RDF and JSON. Since, the primary goal of SBOL is data/ information exchange/ sharing on the web we are using RDF. SBOL utils also include the methods for exchanging data with other common data formats, such as GenBank flat files using BioJava. \begin{DoxyAuthor}{Author}

Detailed Description

The SBOL data model's SequenceFeature for RDF and Json.

The objects of this type represent DNA features which describe DnaComponents found in the Library. They are associated with DnaComponents via the position and strand information found in the SequenceAnnotation. They SHOULD have a DnaSequence object which corresponds to the sequence of the DnaComponent starting and ending at the SequenceAnnotation position in the direction given by the strand. DnaComponents are EXPECTED to be found inside a SBOL Library object.

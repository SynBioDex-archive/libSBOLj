Detailed Description

The SBOL data model's DnaSequence for RDF and Json.

DNA Sequence holds either the actual sequence string or a reference pointer, a URI to it. The SBOL data model is focused on the description of these DNA sequences as they used in assembly of new synthetic biological systems. Information specifying the exact base pair sequence of DNA components and Sequence Features is very important for the ability to replicate synthetic biology work. Both experimental work and theoretical sequence composition research heavily depends on this information.

Developer Notes libSBOLj v.04

Checkout Instructions

  1. Install Git
  2. Clone the project with Git by running In a directory of your choosing 
    $ git clone git@github.com:mgaldzic/libSBOLj.git
    or 
    $ git clone git://github.com/mgaldzic/libSBOLj

Netbeans Project

libSBOLj is being developed in Netbeans. Therefore if you;re using Netbeans you can use the netbeans project directly

  1. Open the libSBOLj project in Netbeans (the libSBOLj root dir)
  2. Clean and Build
  3. run tests

If for some reason that fails

  1. Check the project properties working directory setting. It should be null.
  2. Check that private.properties doesnt include this path /code user.properties.file=C:\Users\mgaldzic\.netbeans\6.9\build.properties /endcode
